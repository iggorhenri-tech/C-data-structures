## 🚀 About the Project

My focus is on mastering Pointer Arithmetic and the memory lifecycle (Heap vs. Stack). By manually managing every byte with malloc, I aim to develop a "mechanical sympathy" for how software interacts with hardware, ensuring zero memory leaks and maximum performance.
